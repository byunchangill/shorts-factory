# 🐶 강아지 쇼츠 대본 템플릿

## 사용 조건
- contentType: `dog`
- 총 씬 수: **6~7개**
- 총 길이: **50~60초** (씬당 평균 8~10초)
- 영상화: 필요 (Ken Burns 또는 AnimateDiff)

---

## 대본 생성 지침

### 오마주 원칙
- 레퍼런스 영상에서 **추상적 패턴만** 추출: 감정 곡선, 훅 유형, 페이싱
- 원본 문장·사건 순서·캐릭터·에피소드는 완전 재창작
- 유사도 검증 기준: phraseOverlap < 0.3, structureOverlap < 0.7

### 훅 패턴 (scene_01)
```
[emotional_shock]  "이 강아지가 혼자서 XX를 해냈다"
[curiosity]        "이 강아지가 병원에서 기다린 이유를 아무도 몰랐습니다"
[question]         "당신이라면 이 상황에서 어떻게 했을까요?"
[confession]       "저는 이 영상을 보고 눈물을 흘렸습니다"
```

### 감정 곡선 구조
```
scene_01 [훅]       → 궁금증/감동 유발 → 감정: surprise / curiosity
scene_02 [배경]     → 상황 설명       → 감정: calm / neutral
scene_03 [전개]     → 사건 발전       → 감정: tension / concern
scene_04 [절정]     → 최고 긴장/감동  → 감정: peak_emotion
scene_05 [해결]     → 따뜻한 마무리   → 감정: warmth / relief
scene_06 [CTA]      → 참여 유도       → 감정: friendly / engaging
```

### 나레이션 스타일
- 문장 길이: **15~25자** (TTS 10초 기준)
- 어조: 따뜻하고 친근한 3인칭 서술
- 금지 표현: "기적", "완벽", "천재 강아지" (과장)
- 권장 표현: "~했다고 합니다", "놀랍게도", "그 순간"


## 전체 대본 예시 구조

```
hook: "이 강아지가 혼자서 버스를 탄 이유"
scenes:
  01 - 훅: 강아지가 혼자 버스 정류장에 서 있는 장면
  02 - 배경: 강아지 이름, 주인과의 관계 소개
  03 - 전개: 혼자 이동하게 된 사건
  04 - 절정: 목적지에 도착해서 발견한 것
  05 - 해결: 따뜻한 재회 또는 감동 결말
  06 - CTA: "이 강아지의 이야기가 감동적이라면 좋아요 부탁드려요"
cta: "구독과 좋아요로 더 많은 강아지 이야기를 만날 수 있어요"
```

## 제목 후보 생성 규칙
- 3개 이상 제시
- 형식: `[숫자/상황] + [감정 키워드] + [결과/반전]`
- 예시: "혼자 병원을 찾아간 강아지의 사연", "이 강아지가 매일 그 자리에 서있는 이유"
- 금지: 원본 제목 패턴 재활용, 과도한 클릭베이트

## 썸네일 훅 후보
- 2~3개
- 이미지 한 장에 들어갈 강렬한 문구 (10자 이내)
- 예시: "이유가 있었다", "혼자였지만", "기다렸어요"
