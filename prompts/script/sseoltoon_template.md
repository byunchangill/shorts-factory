# 📖 썰툰 대본 템플릿

## 사용 조건
- contentType: `sseoltoon`
- 총 씬 수: **7~9개**
- 총 길이: **55~75초** (씬당 평균 7~9초)
- 영상화: **불필요** (이미지 + 자막만)
- 특이사항: 말풍선/텍스트를 이미지에 넣지 않음 → 자막으로 처리

---

## 대본 생성 지침

### 오마주 원칙
- 레퍼런스에서 추출: 훅 패턴, 반전 위치, 페이싱, 컷 전환 속도
- 재창작 대상: 등장인물, 구체적 에피소드, 대화, 배경 설정
- 실존 인물 특정 절대 금지 / 특정 집단 비방 금지

### 훅 패턴 (scene_01)
```
[confession]    "제가 실제로 겪은 일인데요"
[shock]         "이걸 보고 진짜 놀랐습니다"
[question]      "이런 상황이라면 여러분은 어떻게 하실 건가요?"
[number]        "딱 3가지 이유만 말씀드릴게요"
```

### 감정/페이싱 곡선 구조
```
scene_01 [훅]       → 강한 첫 인상     → 속도: fast
scene_02 [배경 설명] → 상황 세팅        → 속도: medium
scene_03 [사건 1]   → 갈등 시작        → 속도: medium
scene_04 [사건 2]   → 갈등 심화        → 속도: fast (긴장감)
scene_05 [절정]     → 최대 반전 직전    → 속도: slow (숨 참기)
scene_06 [반전]     → 예상치 못한 결말  → 속도: fast
scene_07 [해소]     → 감정 정리        → 속도: medium
scene_08 [CTA]      → 참여 유도        → 속도: fast
```

### 나레이션 스타일
- 문장 길이: **12~22자** (빠른 TTS 기준)
- 어조: 1인칭 구어체 ("~했어요", "~거든요", "진짜로")
- 속도감: 짧은 문장 연속 → 긴장감 유발
- 반전 전 씬: 문장 일부러 짧게 ("그런데...", "근데 있잖아요")
- 금지: 실존 지명·기관 특정, 혐오 표현

### 컷 전환 스타일
- 감정 변화 시: **강한 컷** (갑자기 다른 장면)
- 긴장 유지 시: **점진적 전환** (같은 캐릭터 다른 표정)
- 반전 씬: **빈 프레임 → 새 씬** (0.5초 블랙)

## 전체 대본 예시 구조

```
hook: "퇴근길에 이상한 사람을 만났습니다"
scenes:
  01 - 훅: 긴장된 표정의 주인공이 거리에서 누군가와 눈이 마주침
  02 - 배경: 야근 후 지하철역 근처, 조용한 골목
  03 - 사건1: 낯선 사람이 말을 걸어옴
  04 - 사건2: 상황이 이상하게 흘러감
  05 - 절정전: 주인공이 어떡해야 할지 모르는 순간
  06 - 반전: 알고 보니 예상과 전혀 다른 이유
  07 - 해소: 오해가 풀리고 의외의 따뜻한 결말
  08 - CTA: "이런 경험 있으신 분 댓글로 알려주세요"
cta: "공감되셨다면 좋아요와 댓글 부탁드려요!"
```

## 제목 후보 생성 규칙
- 3개 이상 제시
- 형식: `[상황 설정] + [반전 암시]` 또는 `[감정 표현] + [결말 힌트]`
- 예시: "퇴근길 이상한 사람 실화썰", "저만 이런 경험 한 건가요?"
- 금지: 특정 직업·지역·집단 비방, 원본 제목 패턴 복사

## 썸네일 훅 후보
- 강한 반전 암시 문구 (8자 이내)
- 예시: "이게 실화?", "반전있음", "제가 울었어요"
