# 💊 건강 쇼츠 대본 템플릿

## 사용 조건
- contentType: `health`
- 총 씬 수: **6~8개**
- 총 길이: **50~65초** (씬당 평균 8~10초)
- 영상화: 필요 (인포그래픽 스타일)
- 컴플라이언스: **필수** (checkScript 통과 후 생성 완료)

---

## 대본 생성 지침

### 컴플라이언스 우선 원칙
모든 나레이션은 `config/health_compliance.json` 규칙을 통과해야 한다.

**허용 표현 (L1~L2)**
```
"~에 도움이 될 수 있습니다"
"일반적으로 알려진 바로는"
"~라는 연구 결과가 있습니다"
"개인차가 있을 수 있습니다"
"~를 지원할 수 있다고 합니다"
```

**금지 표현 → 자동 재작성 대상 (L4)**
```
❌ "완치", "100% 효과", "기적의 치료"
❌ "의사가 숨기는", "제약회사가 감추는"
❌ "당신은 XX입니다" (진단형)
❌ "반드시 XX하세요", "지금 당장 XX하세요"
❌ "기적", "만병통치", "최강", "놀라운 효과"
```

### 훅 패턴 (scene_01)
```
[number]     "오늘 알려드릴 XX 3가지 습관"
[question]   "이것 매일 하고 계신가요?"
[curiosity]  "XX 드시기 전 꼭 알아야 할 것"
[benefit]    "XX 하나로 바꿀 수 있는 것들"
```

### 정보 전달 구조
```
scene_01 [훅]        → 핵심 주제 제시   → 시청 이유 제공
scene_02 [문제 제시] → 공감 유도        → "이런 증상이 있다면"
scene_03 [원인 설명] → 신뢰 확보        → 출처 T1~T2 인용
scene_04 [핵심 정보] → 주요 효능/방법   → 구체적 수치 가능 (출처 명시)
scene_05 [보충 정보] → 추가 팁/주의사항 → "단, 개인차가 있습니다"
scene_06 [요약]      → 핵심 3줄 정리    → 텍스트 강조 이미지
scene_07 [면책 문구] → 자동 삽입        → "본 영상은 일반적인 건강 정보..."
scene_08 [CTA]       → 참여 유도        → "궁금한 점은 댓글로"
```

### 나레이션 스타일
- 문장 길이: **18~28자** (설명형이라 약간 긴 편)
- 어조: 차분하고 신뢰감 있는 3인칭 정보 전달형
- 숫자/수치: 출처와 함께 ("XX대학 연구에 따르면 OO% 감소")
- 금지: 실제 얼굴 사용, 진단형 문장, 단정적 주장

### 출처 표기 규칙 (sourceCredibility)
```
T1 (WHO, 식약처, 대한의학회): 직접 인용 가능
T2 (병원): "~에 따르면" 접두사 필수
T3 (일반 미디어): "일반적으로 알려진 바로는" 필수
T4 (출처 불명): 사용 금지
```

## 전체 대본 예시 구조

```
hook: "비타민C, 이렇게 드시면 효과가 두 배입니다"
scenes:
  01 - 훅: 비타민C 아이콘과 함께 주제 제시
  02 - 문제: 비타민C를 먹어도 효과 없다고 느끼는 경우
  03 - 원인: 흡수율 차이 (식품의약품안전처 자료)
  04 - 핵심: 식사 후 복용 vs 공복 복용 비교
  05 - 보충: 함께 먹으면 좋은 것 / 피해야 할 것
  06 - 요약: 3가지 핵심 포인트 텍스트 정리
  07 - 면책: "본 영상은 일반적인 건강 정보입니다" (자동 삽입)
  08 - CTA: "궁금한 건강 정보 댓글로 알려주세요"
cta: "구독하시면 더 많은 건강 정보를 알려드립니다"
```

---

## 면책 문구 자동 삽입 규칙
- `scene_07` (마지막 정보 씬 다음)에 항상 삽입
- 나레이션: `"본 영상은 일반적인 건강 정보를 다루고 있습니다. 개인차가 있을 수 있으며, 전문의 상담을 권장합니다"`
- YouTube 설명란: `"⚠️ 본 영상은 의학적 조언이 아닌 일반 정보 제공 목적입니다. 건강 관련 결정은 반드시 전문의와 상담하세요."`

## 제목 후보 생성 규칙
- 3개 이상 제시
- 형식: `[효능/주제] + [방법/조건]` 또는 `[문제] + [해결 암시]`
- 예시: "비타민C 제대로 먹는 방법", "이것만 알면 면역력이 달라집니다"
- 금지: "완치", "기적", "의사가 싫어하는", 단정적 수치

## 썸네일 훅 후보
- 신뢰감 있는 문구 (10자 이내)
- 예시: "이렇게 드세요", "몰랐던 사실", "꼭 보세요"
